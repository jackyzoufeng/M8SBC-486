|----------------------------------------| 0x500000
|          Extended RAM (384KB)          |

|                                        |
|           Extended RAM (3MB)           |

|----------------------------------------| 0x100000 (1MB)
|               BIOS ROM                 |
|----------------------------------------| 0xF0000

|                                        |
